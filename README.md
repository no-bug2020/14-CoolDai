# 酷代
## 简介
CoolDai可以帮助企业在线雇用专业人才，并最终顺利完成项目工作。与此同时，CoolDai也为个人和团队及外包公司提供了一个没有时间地点限制的找项目平台。网站另外也提供了很多非常好的工具来帮助发单方和接单方顺利地完成项目工作和费用结算。且本系统为每位发单方提供客服，解决基本的纠纷。
## 开发使用环境
