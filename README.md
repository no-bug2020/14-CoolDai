# 酷代
## 简介
受疫情的影响，对于每个人来说，多多少少都有接私活的想法，尤其是对于程序员、设计师们。我们的系统为需求发布者与接单者提供了这么一个中间平台，通过本系统可以更加方便、更加快捷的发布需求、接单，且本系统为每位需求发布者提供客服，解决基本的纠纷，为发布需求者与接单者提供一个中间平台。使用Vue+ElementUI构建。
## 开发使用环境 
nodejs、npm、建议使用HBuildX直接编译运行
